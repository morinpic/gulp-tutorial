001_ejs
========
##Usage

クローン後、作業ディレクトリに移動
```
$ git clone https://github.com/morinpic/gulp-tutorial.git
$ cd gulp-tutorial/01_ejs
```

Gulpのインストール  
`-g`オプションを付けてグローバルインストール
```
$ npm install -g gulp
```

開発環境に必要なモジュールをインストール
```
$ npm install
```

開発サーバ起動
-------
```
$ gulp server
```
